# 单词星：21天倒计时 — 用户使用指南

## 项目简介

**单词星：21天倒计时** 是一款将英语单词学习与科幻冒险故事相结合的创新应用。在 21 天的倒计时中，你将扮演一位踏上单词星的冒险者，每天学习 10 个英语单词，同时沉浸在一个关于祖父、记忆与告别的温暖故事中。

- **总词库**：500 个核心英语词汇（中高级难度）
- **学习周期**：21 天，每天 10 个新单词
- **学习方式**：故事剧情 + 交互式学习 + 艾宾浩斯间隔复习

## 注册与登录

首次打开应用会进入登录页面，提供两种方式：

### 方式一：账号登录（推荐）

1. 点击 **「立即注册」** 创建账号
2. 填写用户名、邮箱、密码
3. 注册成功后自动登录
4. **优势**：数据云端同步，换设备不丢失

### 方式二：游客模式

1. 在登录页面点击 **「游客模式（数据存本地）」**
2. 系统自动生成临时账号，直接进入游戏
3. **注意**：数据仅保存在当前浏览器，清除浏览器数据会丢失进度

## 游戏流程概览

整个应用的流程如下：

```
登录 → 序幕剧情 → 每日循环（共21天） → 尾声
                  ┌─────────────────────────┐
                  │  早晨剧情 → 学习单词 → 晚间剧情  │
                  └─────────────────────────┘
```

### 每日流程

每天分为三个阶段：

| 阶段 | 内容 | 说明 |
|------|------|------|
| **早晨剧情** | 推进故事情节 | 了解单词星的世界观和角色 |
| **学习单词** | 学习 10 个新单词 | 核心学习环节 |
| **晚间剧情** | 当日故事收尾 | 回顾一天的收获 |

## 序幕

首次进入游戏时，你将体验一段序幕剧情：

- 主角驾驶 Promise 号飞船在宇宙航行
- 检测到与祖父失踪航线吻合的引力波
- 追踪信号来到单词星
- 遇到神秘的 Lexicon，开启 21 天学习挑战

**序幕完成后**，进入正式的学习循环。

## 学习单词

### 学习界面

学习页面展示当天的单词，每个单词以「单词居民」的形象呈现：

1. **看到单词**：屏幕中央显示英语单词
2. **判断认识与否**：
   - 点击 **「认识」** — 该单词从学习列表中永久移除（不影响学习计数）
   - 点击 **「不认识」** — 展示单词的详细含义和例句
3. **查看含义**：点击后显示：
   - 中文释义
   - 英文例句
4. **确认记住**：点击 **「我记住了」** 完成该单词的学习

### 每日目标

- 每天需要学习 **10 个新单词**
- 进度条显示当前学习进度
- 完成后自动进入晚间剧情

### 单词状态

| 状态 | 含义 |
|------|------|
| `new` | 新单词，未学习 |
| `learning` | 学习中，需要复习 |
| `reviewing` | 复习中 |
| `mastered` | 已掌握 |

## 复习系统

### 间隔复习机制

应用采用 **艾宾浩斯遗忘曲线** 原理，自动安排复习计划：

| 复习次数 | 复习间隔 |
|----------|----------|
| 第 1 次 | 1 天后 |
| 第 2 次 | 2 天后 |
| 第 3 次 | 4 天后 |
| 第 4 次 | 7 天后 |
| 第 5 次+ | 已掌握，不再安排复习 |

### 复习页面

当有需要复习的单词时，进入复习页面：

1. 显示英语单词
2. 从 **4 个选项** 中选择正确的中文含义
3. 选择后立即显示正确/错误反馈
4. 点击 **「我记住了」** 进入下一个单词
5. 复习完成后显示统计（正确数、错误数、正确率）

## 数据存储

### 游客模式

- 学习进度保存在浏览器 `localStorage`
- 键名：`wordstar_game_state`
- 清除浏览器数据会导致进度丢失

### 账号模式

- 进度通过后端 API 同步到服务器
- 支持 `accessToken` + `refreshToken` 认证
- 换设备登录后自动恢复进度

## 常见问题

### Q: 如何重新开始游戏？

在个人中心页面，找到「调试工具」区域，点击「重置游戏」。

### Q: 忘记密码怎么办？

目前需要联系管理员重置密码，或重新注册账号。

### Q: 游客模式可以转为正式账号吗？

游客模式数据仅保存在本地，无法迁移到正式账号。建议一开始就注册账号使用。

### Q: 每天必须完成学习吗？

不强制。你可以在任意时间继续学习，系统会记住你的进度。但连续学习天数会影响成就解锁。

### Q: 认识的单词会怎样？

点击「认识」的单词会从学习队列中永久移除，不影响当天学习计数。系统会认为你已经掌握了该单词。

### Q: 如何复习之前学过的单词？

在主页点击「复习单词」或在每日学习流程中，系统会自动安排需要复习的单词。复习基于艾宾浩斯遗忘曲线，间隔为 1-2-4-7 天。
