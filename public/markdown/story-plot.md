# 📖 单词呕吐机 - 完整故事剧情设计文档

> **项目名称**：单词星：21天倒计时
> **故事类型**：科幻、冒险、成长、哲学
> **核心主题**：记忆、存在、连接、承诺
> **目标用户**：英语学习者
> **设计日期**：2025年

---

## 🌟 故事概述

**一句话梗概：**
一位来自2035年的宇宙探险家迫降在神秘的"单词星"，遇到了孤独的守护者Lexicon，必须在21天内通过认知210个单词来拯救这个垂危的文明，最终面临关于记忆、承诺和归属的终极选择。

**核心冲突：**
- 外部冲突：单词星即将消亡，倒计时21天
- 内在冲突：对家园的思念 vs 对新朋友的牵挂
- 道德冲突：完成任务离开 vs 拯救这个文明

## 🌍 世界观设定

### 单词星（Word Star）

#### 基本信息
- **位置**：半人马座旋臂，未知扇区
- **外貌**：紫色星球，地表覆盖发光晶体
- **大气**：可呼吸，淡紫色天空
- **重力**：与地球相似（0.9G）
- **自转周期**：26小时

#### 特殊现象
```
【能量衰减】
- 星球核心能量正在耗尽
- 倒计时：21天
- 原因：长期没有被足够的文明认知

【单词居民】
- 外形：机器人般的小型实体
- 胸口有显示屏，显示一个单词
- 有自我意识和情感
- 需要"被认知"才能维持存在

【空白化】
- 当单词长期不被认知
- 会逐渐失去能量
- 最终变成"空白单词"
- 状态：永久死亡
```

#### 历史背景
```
【800年前】
- 记忆族文明创造单词星
- 目的：保存所有文明的语言遗产
- 方法：词汇具象化技术

【700年前】
- "大遗忘"事件席卷宇宙
- 所有文明失去语言能力
- 记忆族为了让词汇存活
- 将他们发射到宇宙深处

【600-100年前】
- Lexicon作为守护者
- 带着单词流浪宇宙400年
- 尝试100多个星球
- 没有文明愿意接纳"活着的单词"

【100年前】
- 建立单词星作为庇护所
- 开始等待下一个文明
- 4个21年周期过去
- 前4位探险家都选择了离开

【现在】
- 最后一个21天周期
- 第5位探险家（用户）的到来
- 剩余时间：21天
```

## 👥 主要角色

### 用户（主角）
- **身份**：地球联合星际探险队第42号成员
- **年龄**：28岁
- **职业**：深空探险家
- **动机**：寻找适合人类殖民的星球
- **特殊**：祖父曾是50年前的访客
- **成长**：从交易心态到真正连接

### Lexicon（单词吞吐机）
- **身份**：单词星守护者、宇宙保险创始人
- **年龄**：约800岁（实际运行时间）
- **种族**：半机械能量体
- **使命**：守护词汇文明直到被认知
- **性格**：理性孤独、外表幽默内心脆弱
- **秘密**：等待一个不会离开的人

> 📌 **详细角色设定**：见《人物塑造文档》

### HOPE（希望）
- **身份**：最古老的单词居民之一
- **特质**：挑剔、选择性强
- **意义**：代表人类最需要的品质
- **作用**：最终选择的关键引导者

### ABANDON（抛弃）
- **身份**：第一个被用户认知的单词
- **特质**：忧郁、害怕被遗忘
- **象征**：800年来被遗忘的创伤
- **成长**：从恐惧到信任

## 🎨 故事设计理念

### **核心主题**
```
1. 记忆与存在
   "我们被记住，所以我们存在。"

2. 连接 vs 交易
   从"等价交换"到"真正的情感连接"

3. 承诺的传承
   祖父的承诺通过你延续

4. 家的定义
   家不是一个地方，而是"被记住的地方"
```

### **情感弧线**
```
Day 1-3:  好奇、困惑
Day 4-7:  发现、震惊、责任
Day 8-14: 思考、转变、决心
Day 15-20: 危机、牺牲、准备
Day 21:   选择、结局
```

### **角色成长**
```
起点：交易心态
- "我做这个是为了燃料"
- "单词是工具"

转折：情感连接
- "他们是有生命的"
- "我不能让他们死"

终点：真正的守护
- 无论哪个结局
- 都是从"交易"到"责任"的转变
```
